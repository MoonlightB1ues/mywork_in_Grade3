# Mywork_in_大三
包括大三的编程作业和大数据,数值分析,金融经济的笔记
## 使用方法
点击右上角绿色按钮里的下角标,下载压缩包
解压后将代码复制到你对应的环境里
如果是pycharm,会在运行的时候自动帮你安装宏包
## 注意
课程笔记我用obsidian写的,github渲染不出来,如果有兴趣,可以下一个obsidian软件,把笔记导入即可查阅.

大数据的第一个编程作业,由于体积问题原始数据没有上传,可以自行去学习通下载到本地,存在data里面应该就能跑了.
